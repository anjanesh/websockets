# websockets

Install websockets

`pip install websockets`
or
`pip3 install websockets`

run serverp.py as :

`python3 server.py`
or
`python server.py`
or
`py.exe server.py`

Open client.html in Visual Studio Code and click on Live Preview on the bottom right and it should fire up client.html in the browser
